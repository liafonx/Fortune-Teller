# Fortune Teller

将卡牌的随机结果显示在悬停信息弹窗中。

## 功能

- 将卡牌介绍文本移动到独立的弹窗。
- 主信息面板改为卡牌随机结果预览区域。
- 收藏界面保持原版弹窗（不启用预测覆盖）。
- 悬停预测时保持随机状态不被消耗。

## 当前支持

- 命运之轮
- 皇帝、女祭司（始终显示完整生成结果，不受当前消耗牌槽位限制）
- 审判、灵魂、幽灵
- 隐形小丑
- 光环、符印、占卜、妖法、灵质、生命十字章
- 使魔、严峻、咒语、火祭

## 配置项

- 卡牌页签：按支持卡牌逐项启用或禁用替换弹窗。
- 显示页签：开关主弹窗卡牌名称、卡牌介绍小弹窗、卡牌类型标签。
- 日志页签：开关 info/debug 级别日志。
