# Fortune Teller

悬停即可预览卡牌效果——使用前就能看到结果。

## 功能介绍

- 悬停支持的卡牌即可预览生成或作用目标。
- 卡牌效果说明移至侧边小弹窗。
- 收藏界面保持原样。
- 悬停不影响游戏——随机结果不会改变。

## 支持卡牌

**塔罗牌**
- 命运之轮（显示目标小丑，失败则显示"Nope!"）
- 皇帝、女祭司（显示将要生成的所有卡牌）
- 审判、灵魂、幽灵

**幻灵牌**
- 光环、符印、占卜、妖法、灵质、生命十字章
- 使魔、严峻、咒语、火祭

**小丑牌**
- 8号球、错印小丑、疯狂、乌合之众、幻觉
- 流浪者、叠加态、占卜师、第六感、通灵、证书、帕奇欧
- 隐形小丑
- 悬停蓝图或头脑风暴时显示被复制小丑的预测结果

**蜡封**
- 紫蜡蜡封（高亮手牌可预览生成的塔罗牌结果）

## 设置

在 Mod 设置页签中：
- **卡牌** — 开关塔罗/幻灵预测；开关紫蜡手牌预览。
- **小丑** — 开关小丑预测；特殊选项包含隐形小丑提前显示和错印下一张抽牌预览。
- **显示** — 显示/隐藏卡牌名称、效果弹窗和标签；切换在触发阶段外显示预测。
- **日志** — 启用详细日志用于排查问题。
