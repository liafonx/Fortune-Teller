# Fortune Teller

悬停即可预览卡牌效果——使用前就能看到结果。

## 功能介绍

- 悬停支持的卡牌即可预览生成或作用目标。
- 卡牌效果说明移至侧边小弹窗。
- 收藏界面保持原样。
- 悬停不影响游戏——随机结果不会改变。

## 支持卡牌

- 命运之轮（显示目标小丑，失败则显示"Nope!"）
- 皇帝、女祭司（显示将要生成的所有卡牌）
- 审判、灵魂、幽灵
- 隐形小丑
- 光环、符印、占卜、妖法、灵质、生命十字章
- 使魔、严峻、咒语、火祭
- 紫蜡封印（高亮手牌可预览下一张塔罗结果）

## 设置

在 Mod 设置页签中：
- **卡牌** — 按卡牌开关预测，切换紫蜡预览和隐形小丑提前显示。
- **显示** — 显示/隐藏卡牌名称、效果弹窗和标签。
- **日志** — 启用详细日志用于排查问题。
