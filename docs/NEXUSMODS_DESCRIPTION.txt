[center][size=5][b]Fortune Teller[/b][/size]

[i]Preview deterministic random outcomes directly in hover UI.[/i][/center]

Fortune Teller surfaces random-result card outcomes before use/sell by showing a visual forecast in the card info panel.

[list]
[*]Effect text moved to a dedicated popup
[*]Main panel replaced with forecast card previews
[*]Stable card sizing with dynamic width for multi-card outcomes
[*]Collection view keeps vanilla info panel behavior
[*]RNG-safe hover prediction (no random state consumption)
[*]Built-in config tabs for per-card toggles, display options, and debug/info logs
[/list]

[list]
[*]Wheel of Fortune
[*]The Emperor, The High Priestess (always show full generated results, ignoring current consumable slots)
[*]Judgement, The Soul, Wraith
[*]Invisible Joker
[*]Aura, Sigil, Ouija, Hex, Ectoplasm, Ankh
[*]Familiar, Grim, Incantation, Immolate
[/list]
